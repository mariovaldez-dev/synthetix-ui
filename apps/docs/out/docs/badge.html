Badge

Etiqueta compacta para mostrar estado, categorías o contadores.

Instalación

bash
npx @synthetix-ui add badge

Variantes

DefaultPrimarySuccessWarningDangerOutline

Tamaños

SmallMediumLarge

Personalización

Usa className para crear badges con colores, bordes o tipografía personalizada.

VioletGradientMONOBorderedTeal
tsx
<Badge className="bg-violet-100 text-violet-700 border-violet-200">Violet</Badge>
<Badge className="bg-gradient-to-r from-pink-500 to-rose-500 text-white border-transparent">Gradient</Badge>
<Badge className="rounded-sm font-mono uppercase bg-zinc-900 text-zinc-100 border-transparent">MONO</Badge>

Props

PropTipoDefaultDescripción
variant"default" | "primary" | "success" | "warning" | "danger" | "outline""default"Color del badge
size"sm" | "md" | "lg""md"Tamaño del badge

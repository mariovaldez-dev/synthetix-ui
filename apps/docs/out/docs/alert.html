Alert

Documentación para el componente Alert. Próximamente se añadirán ejemplos de uso y props detallados.

Instalación

bash
npx @synthetix-ui add alert

Preview

Básico

Heads up!
You can add components to your app using the cli.

Variantes

Note
This is an informational alert.
Success
Action completed successfully.
Warning
Please check your credentials.
Error
Something went wrong.

Personalización

Usa className para personalizar estilos individuales. Para cambiar el color primario globalmente, usa el selector de tema en la barra superior.

Personalizado Violet
Alerta con colores fuera de las variantes predefinidas.
Estilo borde lateral
Bordes izquierdos son un patrón común en alertas.

Uso básico

tsx
import { Alert } from "@synthetix-ui/core";

export function Example() {
  return <Alert />;
}

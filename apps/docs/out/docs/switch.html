Switch

Toggle booleano accesible implementado sobre input[type=checkbox] con animación CSS.

Instalación

bash
npx @synthetix-ui add switch

Uso

tsx
import { Switch } from "@/components/ui/switch";

export function Example() {
  return <Switch label="Modo oscuro" />;
}

Demo

Recibirás ofertas cada semana

tsx
<Switch label="Modo oscuro" />
<Switch label="Notificaciones activas" defaultChecked />
<Switch label="Opción deshabilitada" disabled />
<Switch
  label="Correos de marketing"
  helperText="Recibirás ofertas cada semana"
  size="lg"
/>

Personalización

Usa className para personalizar estilos individuales. Para cambiar el color primario globalmente, usa el selector de tema en la barra superior.

tsx
<div style={{ "--color-primary": "263 70% 50%" }}>
  <Switch label="Violet" defaultChecked />
</div>
<div style={{ "--color-primary": "347 77% 50%" }}>
  <Switch label="Rose" defaultChecked />
</div>
<div style={{ "--color-primary": "160 84% 38%" }}>
  <Switch label="Emerald" defaultChecked />
</div>

Props

PropTipoDefaultDescripción
labelstring—Etiqueta visible del switch
checkedboolean—Estado controlado
defaultCheckedboolean—Estado inicial no controlado
onChange(e: React.ChangeEvent<HTMLInputElement>) => void—Callback al cambiar de estado
disabledbooleanfalseDeshabilita el switch

Implementación

Usa un <input type="checkbox" class="peer sr-only"> oculto con un elemento hermano estilizado via clases peer-checked: de Tailwind. Esto garantiza accesibilidad nativa sin JavaScript adicional.

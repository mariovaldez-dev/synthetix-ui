Spinner

Indicador de carga animado con variantes de tamaño y color.

Instalación

bash
npx @synthetix-ui add spinner

Uso

tsx
import { Spinner } from "@/components/ui/spinner";

export function Example() {
  return <Spinner size="md" color="primary" />;
}

Tamaños

Cargando...Cargando...Cargando...Cargando...Cargando...
tsx
<Spinner size="xs" />
<Spinner size="sm" />
<Spinner size="md" />
<Spinner size="lg" />
<Spinner size="xl" />

Personalización

Usa className para personalizar estilos individuales. Para cambiar el color primario globalmente, usa el selector de tema en la barra superior.

Cargando...Cargando...Cargando...Cargando...
Cargando...
tsx
<Spinner className="text-violet-500" size="md" />
<Spinner className="text-rose-500" size="md" />
<Spinner className="text-emerald-500" size="md" />
<Spinner className="text-amber-500" size="lg" />

PropTipoDefaultDescripción

color"primary" | "muted" | "white" | "success" | "danger""primary"Color de la animación
